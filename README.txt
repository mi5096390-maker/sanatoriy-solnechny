# Санаторий Солнечный — многостраничный сайт + оплата по QR (ЮMoney)

## Что внутри
- `index.html` — главная (акции, программы, номера, профили, о санатории, территория, контакты)
- `booking.html` — 2-я страница: расчёт и QR-код для оплаты на ЮMoney-кошелёк

## Как работает оплата
- На `booking.html` рассчитывается сумма:
  - 50% сейчас и 50% при заезде
  - или 100% сейчас со скидкой 20%
- После расчёта сайт генерирует QR-код.
- При сканировании QR открывается страница оплаты ЮMoney с подставленной суммой и назначением платежа.

## Важно
- Без серверной интеграции сайт не сможет автоматически узнать «оплачено».
